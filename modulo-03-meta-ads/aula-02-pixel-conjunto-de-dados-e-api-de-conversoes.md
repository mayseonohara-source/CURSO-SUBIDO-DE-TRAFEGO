 Eventos:  o  que   são  e  para  que  servem?   
 Basicamente,  os  eventos  são  ações  que  o  usuário  realiza  dentro  do  seu  site,  mas  
para
 
que
 
servem?
 
  Os  eventos  servem,  principalmente,  para  duas  coisas:  criação  de  públicos  e  
otimização
 
de
 
campanhas
 
com
 
base
 
nos
 
eventos
 
(ações).
  Com  base  nisso,  existem  3  tipos  de  criação  de  eventos:     
1.  Eventos  padrão:  
eventos  criados  a  partir  da  edição  do  código  do  Pixel,  ou  seja,  manualmente  inserir  eventos  neste  código  (ou  por  ferramentas  que  você,  porventura,  utilizar).   
2. Eventos  personalizados:   criados  a  partir  da  edição  do  código  do  Pixel,  mas,  neste  caso,  você  define  o  seu  nome  e  edita-o  no  código.  
3.
 Conversões  personalizadas:  eventos  criados  a  partir  de  uma  URL  (você  consegue  criar  públicos  baseados  em  quem  utilizou  determinada  URL).    
   
 Com  essas  informações  em  mãos,  agora  é  só  criar  o  seu  Pixel,  identificar  o  ID

seu
 
código,

de
 
eventos
